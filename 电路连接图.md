## 连接说明表

| ESP32引脚 | 连接到 | 说明 |
|-----------|--------|------|
| GPIO0 | 轻触开关 | 按钮输入（开发板自带Boot按键可复用） |
| 3.3V | 轻触开关另一端 | 按钮供电 |
| GND | 轻触开关GND端 | 按钮接地 |
| GND | TP4056 B- | 电池负极 |
| VIN | TP4056 B+ | 电池正极（通过USB充电后输出） |
| GPIO22 | LED+ (可选) | 状态指示灯 |
| GND | LED- (可选) | LED负极 |

## 接线详细步骤

### 第一步：准备组件
- ESP32开发板 × 1
- 轻触开关（6mm × 6mm）× 1
- TP4056锂电池充电模块 × 1
- 3.7V锂电池（500mAh-1000mAh）× 1
- 杜邦线若干
- 小型LED灯（可选）× 1
- 220Ω电阻（可选，用于LED限流）

### 第二步：连接按钮到ESP32

```
轻触开关    ESP32
  (1) ──────► 3.3V
  (2) ──────► GPIO0
  (3) ──────► GND
```

**注意**：ESP32开发板自带Boot按钮已连接到GPIO0，可直接使用，无需外接按钮。

### 第三步：连接电池系统

```
锂电池 ──────────────► TP4056
  (+)    B+          (+) (红色)
  (-)    B-          (-) (黑色)

TP4056 ──────────────► ESP32
  B+    VIN          (通过USB接口或直接连VIN)
  B-    GND          (通过USB接口或直接连GND)
```

**TP4056充电模块使用说明**：
- 通过USB接口给电池充电
- 充电时LED红灯亮，充满后变绿
- B+输出3.7V-4.2V

### 第四步：连接LED（可选）

```
ESP32 ──────────────► LED
  GPIO22 ──[220Ω电阻]──► LED(+)
  GND    ──────────────► LED(-)
```

## 引脚定义图（ESP32 DevKit V1）

```
                   ┌─────────────┐
                   │    ESP32    │
                   │  DevKit V1  │
                   └─────────────┘
    ┌─────────────────────────────────┐
    │  EN  ┌────┐  VP  VN  34  35  32  │
    │      │USB│  33  25  26  27  14  │
    │  3V3  │   │  12  13   GND  16  17 │
    │       └────┘   4   0   2   15    │
    │  VIN         5  18  19  21  3   22
    │  GND                     1   TX  RX
    └─────────────────────────────────┘
         ▲  ▲
         │  │
        VIN 3.3V
        电池 电源
```

**关键引脚说明**：
- **VIN**：输入5V-12V，可连接TP4056的B+输出
- **3.3V**：输出3.3V，可给按钮供电
- **GND**：公共地线
- **GPIO0**：按钮输入（开发板自带Boot按键）
- **GPIO22**：LED输出（推荐使用PWM引脚）

## 注意事项

1. **电源顺序**：先连接电池，再接ESP32，避免冲击电流
2. **极性检查**：确保电池正负极连接正确，反接会损坏ESP32
3. **充电安全**：使用TP4056充电时，不要短路B+和B+
4. **蓝牙配对**：首次配对需要在小度音箱近处进行（<1米）
5. **音频质量**：确保ESP32音频输出格式与小度音箱兼容（建议16kHz 16bit）
